Django To-Do List App

A simple and efficient To-Do List Web Application built using Django.
This app allows users to manage daily tasks with full CRUD functionality.

 Features
 Add new tasks
 Update/edit tasks
 Delete tasks
 Mark tasks as completed
 View all tasks
 Timestamp for each task
 
🛠️ Tech Stack
Backend: Python, Django
Frontend: HTML, CSS
Database: SQLite
Tools: Git, GitHub
